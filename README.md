## 📦 Git LFS Required

This repository uses **Git LFS** for large assets in `binaries/`, `Tools/`,
`assets/`, and some files under `fuckdl/devices/`.

Before cloning or pulling large-file updates, install Git LFS and initialize it:

```powershell
git lfs install
git clone https://github.com/chromedecrypt/Fuckdl.git
```

If you already cloned the repo before installing Git LFS, run:

```powershell
git lfs install
git lfs pull
```

## 🚀 Quick Start

1. Install **Python 3.10 – 3.12** and add it to `PATH`.
2. Install the **Microsoft Visual C++ Redistributable**:
   <https://aka.ms/vs/17/release/vc_redist.x64.exe>
3. Run **`install.bat`**.
4. Read **[`How.to.use.md`](How.to.use.md)** for the full red & gold setup
   guide (cookies, credentials, CDM device folders, per-service examples).

```powershell
.\install.bat
.\help.bat
.\download.DisneyPlus.bat
```

## 🛠 Common Usage

```powershell
:: Show help
poetry run fuckdl dl -h

:: 1080p, no key cache, just print keys (Disney+ entity ID)
poetry run fuckdl dl -q 1080 --no-cache --keys DSNP entity-XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX

:: Specific season / episode range
poetry run fuckdl dl -q 1080 -w S01E01-S01E05 AMZN <ASIN>
```

| 🔴 Flag | 🟡 Meaning |
|---|---|
| `-q` | Resolution (e.g. `1080`, `2160`) |
| `-v` | Video codec (default `H264`) |
| `-a` | Audio codec |
| `-r` | Color range (`SDR` / `HDR` / `DV`) |
| `-w` | Wanted episodes (`S01E01-S02E03`) |
| `-al` / `-sl` | Audio / subtitle language |
| `--keys` | Print decryption keys only |
| `--cache` / `--no-cache` | Use / bypass key vault |
| `--proxy` | Proxy URI or 2-letter country code |

## 🎨 Red & Gold UI

The CLI now ships with a themed startup banner and colored logs:

- 🟥 **Red** — errors, warnings, the logger name
- 🟨 **Gold** — INFO, success, timestamps, level names
- ⬜ **White-on-red** — critical errors

The banner appears automatically when running any `fuckdl` command. The
`install.bat` and `help.bat` launchers also set the Windows console title
and color to match the theme.
